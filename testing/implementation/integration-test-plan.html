Scheduling Tool Integration Test Plan


Perform integration testing in the following phases:

  1. Integrate the models.data.databases package with the controllers.data.databases package and rerun the class tests in the models.data.databases package.

  2. Add the schedtool package to the integrated models.data.databases + controllers.data.databases package and rerun the class tests in the schedtool and models.data.databases package.

  3. Integrate the models.admin.generation package with the controllers.admin.generation package and rerun the class tests in the models.admin.generation package.

  4. Add the models.admin.generation + controllers.admin.generation integrated packages to the schedtool + models.data.databases + controllers.data.databases integrated packages and rerun the class tests in the models.admin.generation, schedtool, and models.data.databases packages.

  5. Integrate the models.admin.analytics package with the controllers.admin.analytics package and rerun the class tests in the models.admin.analytics package.

  6. Add the models.admin.analytics + controllers.admin.analytics integrated packages to the schedtool + models.data.databases + controllers.data.databases + models.admin.generation + controllers.admin.generation integrated packages, and rerun all the class tests in each package.
