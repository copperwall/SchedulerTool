This design documentation describes the packages of the SchedulerTool softw are system. The system is comprised of two application programs:

  • the SchedulerTool desktop administration program, which allows scheduling administrators to automatically generate schedules, as well view and edit instructor preferences and student votes towards future course options. Administrators can also manage databases from this tool.
  • the SchedulerTool web client, which allows students to view upcoming course schedules and vote on potential courses for upcoming terms. The web client also allows instructors to set their preferences and availability towards upcoming courses.
See the companion SchedulerTool Requirements Specification for a more complete set of details of SchedulerTool functionality.
