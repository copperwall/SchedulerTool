2.2.3 Edit Locations

When the admin clicks the Locations button under the Databases menu, the scheduler program shows the following screen.

Figure 2.2.3-1 shows a list of all locations in the database.

Initially, the location database is empty. To add a location to the list of available locations, the administrator clicks the Add button. The scheduler program displays the following dialog:

Figure 2.2.3-2a shows the location Add/Edit dialog.

Figure 2.2.3-2b shows the filled dialog.

After filling in the information, the administrator clicks on Submit to close the dialog and add the entry to the list of locations.

Figure 2.2.3-3 shows the list of locations after one item is added.

To add another item, the administrator repeats the process for the other location.

Figure 2.2.3-4 shows the list of locations after one more item is added.

After the administrator adds the second location, the table sorts the locations by their building numbers and then by room numbers.

To fill the database, an administrator would need to repeat the process for other locations.
