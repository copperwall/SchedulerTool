1.5.4 Related Systems: RosterBooster

RosterBooster is a Windows application with its own algorithm to "solve" a roster for shifts at a workplace. The software available is test-only, therefore, features are limited. There are examples of rosters in which you can hit Solve and set a time limit, a maximum allotted memory, and others. It's a simple program but the UI isn't the best looking. It's difficult to manage everything and it's easy to get lost. Preferences are very user-unfriendly and it doesn't seem like anyone would use this.

    Good Features:
  • Customizable algorithm parameters.
  • Can pause/play, stop, and cancel during solving.
  • Drag-and-drop works well and notifies when there's a conflict with a constraint.
    Bad Features:
  • Need to have a large monitor to get the most out of it.
  • UI is not the greatest.
  • Constantly have to drag and drop panel boundaries.
  • Not available "on the cloud."
    Missing Features:
  • Ability to archive schedules.
  • Ability to save schedules.
