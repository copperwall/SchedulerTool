1.5.3 Related Systems: RoboRoster

Like FindMyShift, RoboRoster is an online scheduling application. Unlike FindMyShift, this is packed full of features and has a friendly, simple, and powerful user interface.

RoboRoster contains almost all the features the Scheduling Tool offers and more. For the goals of the scheduling tool, however, most of these features are unnecessary. Implementing these features would complicate the scheduling tool, making it more difficult to use.

The main feature missing from this software is the ability to automatically generate schedules.

    Good Features:
  • End users are able to set their availability and shift preferences.
  • End users can submit availibility change requests at any time.
  • The scheduler is alerted when a time, location, or preference conflict arises.
  • Staff and Locations are easily added, removed and customized.
  • New shifts can created and customized inline.
  • Shifts are constrained by employee qualification, department(location) and user availibility.
  • A history of shift edits can be viewed inline.
  • Weekly schedules can be saved as templates.
  • End users can pick up available shifts
  • Schedules can be viewd in both horizontal and vertical orientation.
  • Schedules can be filtered by department.
  • Administrators have full control and can freely modify the schedule.
  • Contains User and location administration sections.
  • Schedule reports can be generated.
    Bad Features:
  • Schedules must be generated weekly.
  • Too many unnecessary features make the interface a bit complicated.
    Missing Features:
  • Automatic schedule generation.
