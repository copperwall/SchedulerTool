1.5 Related Systems

    Widely used commercial systems that provide functionality comparable to the Scheduler Tool include:
  • RoboRoster
  • FindMyShift Online Rosters
  • Optaplanner
  • RosterBooster
    Each of these systems provides a wide variety of staff rostering features. The goal for the scheduling tool is to provide the best, most useful features found in these tools - sort of a compilation of improved features from each. The three core features the scheduling tool improves on, in relation to these systems, are as follows:
  1. streamlined automatic generation of schedules based on hard and soft constraints
  2. simpler, more straightforward methods for end users to input preferences (soft constraints)
  3. more natural interface for manually editing generated schedules

The goals for the Scheduling Tool relating to public domain source code and sound pedagogical development are not met by any of the above commercial systems.
