2.3 Basic Schedule Generation

To generate a schedule, the administrator clicks the Generate menu item. While the program generates the schedule, this loading screen appears (Figure 2.3-1).

Figure 2.3-1: This loading dialog appears while the program is generating a schedule.

Once the program finishes generating, it shows the schedule. An example schedule is shown below (Figure 2.3-2).

Figure 2.3-2: An example generated schedule.

In order to generate a basic schedule, the administrator must first ensure that the course database is not empty. If the course database is empty and the administrator clicks the Generate button, the following error appears (Figure 2.3-3). Refer to the Course Database Management section (2.2.2) to fill in the database and fix this error.

Figure 2.3-3: This error message appears when the administrator tries to generate a schedule with an empty course database.

If the instructors database is empty, the program does not show an error but instead fills in the missing instructors in the schedule with the placeholder STAFF. Similarly if the locations database is empty, the program fills in the schedule with TBA in place of buiding and room numbers.
